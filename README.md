# Algorithm Visualizer

Introduction
------------

The project is to show how algorithm works using animation.

Algorithms
------------

This application currently shows the following algorithms: 

**SHell Sort**

**Bubble Sort**

**Merge Sort**
 
**Quick Sort**

**Heap Sort**

**Depth-first Search**

**Breath-first Search**

**Dijkstra's Search**

**A* Search**

**Binary Search Tree**

**Heap Tree**
